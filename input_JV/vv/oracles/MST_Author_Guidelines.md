# MST Author Guidelines

**Journal:** Measurement Science and Technology (IOP Publishing)
**ISSN:** 0957-0233 (print), 1361-6501 (online)
**Target:** Special Issue - "Data-Centric Exploration and Explanation of Physical and Engineering Phenomena"
**Deadline:** March 31, 2026

---

## Scope Requirements

MST papers must:
1. **Advance measurement science** — not merely apply established techniques
2. **Include uncertainty analysis** — precision/accuracy of all measurements
3. **Demonstrate novelty** — clear statement of what's new (< 100 words)
4. **International relevance** — accessible to global readership

> "All submitted articles should contain consideration of the uncertainty, precision and/or accuracy of the measurements presented."

---

## Article Structure (IMRAD)

| Section | Requirements |
|---------|--------------|
| **Title** | Concise, informative, include key terms |
| **Abstract** | Max 300 words, self-contained, no undefined acronyms |
| **Keywords** | 3-7 keywords required |
| **Introduction** | Background, novelty statement, relevance to measurement science |
| **Methods** | Sufficient detail for reproducibility |
| **Results** | Clear presentation of findings |
| **Discussion** | Significance, comparison with prior work |
| **Conclusion** | Novelty summary, future directions |
| **Acknowledgements** | Funding, conflicts of interest |
| **Data Availability** | Statement required |

## Formatting

### General
- No strict page limit (check specific requirements)
- Single or double column accepted
- LaTeX (iopart.cls) or Word accepted
- Consistent spelling/hyphenation throughout

### Mathematics
- Punctuate equations as part of sentences
- Roman for: d (differential), e (exponential), i (complex), trig functions
- Units: m s⁻¹ format with space between value and unit
- Numbers: thin space every 3 digits (15 000)

### Figures
- Vector format preferred (EPS or PDF)
- Text size: 8-12 pt at final size
- Avoid color as sole information carrier
- Sequential numbering
- Multi-part: label as (a), (b), etc.
- Caption must be self-explanatory

### Tables
- Place at page top
- Number sequentially
- Footnotes: lowercase superscript letters

### References
- Vancouver (numerical) OR Harvard (alphabetical) — be consistent
- Include DOIs when available
- Complete bibliographic information required

---

## Data Availability

- Supplementary files: up to 50 MB each (150 MB total)
- Large datasets: use external repository (Zenodo, Figshare, etc.)
- Unique DOIs assigned to supplementary material
- Data availability statement required
